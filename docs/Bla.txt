

The main screen contains user controls to create a random song, and to pause, play, and stop playing the song. The song generator creates a song based on the music theory and several input parameters. Each song consists of verses, which are made up of phrases, which are made up of measures. The song is composed over several parts which arranges notes based on the generated harmony, melody, and rhythm. The notes are translated into a grid which is displayed by a grid screen.
